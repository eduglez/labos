Las peticiones de apoyo técnico serán realizadas por los IP de los proyectos o aquel personal investigador autorizado por el primero.

No se podrá realizar una nueva petición de apoyo técnico para un mismo proyecto hasta que la petición anterior haya sido finalizada, decir que haya sido firmada su finalización por el solicitante.

Los investigadores no incluidos en ningún proyecto serán atendidos de forma rotatoria, siendo considerados en su totalidad como un único grupo.

En la petición se deberá especificar el conjunto de tareas solicitadas. Las mismas no podrán exceder el equivalente a 2 días de trabajo del técnico. El técnico responsable valorará el tiempo necesario para realizar las tareas de la petición y no admitirá solicitudes que impliquen una carga superior de trabajo. En dicho caso, el usuario deberá disminuir el número de las tareas y/o muestras para ajustarse la carga de trabajo establecida. Existe una tabla para cada unidad que indica el número de tareas máximo por petición, la cual se utilizará para delimitar el número de tareas en una petición de forma automática.

Las eventuales peticiones de análisis con pocas tareas podrán tratarse de forma prioritaria a decisión del técnico.

Las muestras, que deberán ir claramente identificadas, se entregarán como un lote completo al técnico. Para no retrasar el trabajo, no se modificará el contenido del lote una vez iniciado el mismo.
